# index-dezhoupoker-notes

这是一个用于归档和发布多个独立 HTML 页面的仓库。仓库内包含各类静态网页资源，主要用于保存和展示不同主题的页面内容，不针对任何特定域名或网站。

## 项目简介

本项目旨在以仓库形式统一管理多个独立的 HTML 页面，方便历史内容的存档、版本追踪和访问。页面内容可能涵盖笔记、工具、参考信息等，所有页面均为静态 HTML 文件，无需后端支持即可直接浏览。

## 目录说明

- `/` — 根目录，包含主索引页面或初始页面
- `pages/` — 存放主要 HTML 页面文件的目录（如存在）
- `assets/` — 存放页面所需的样式、脚本、图片等资源文件（如存在）
- `archive/` — 存放历史版本或不再更新的页面（如存在）

> 实际目录结构可能随仓库更新而变化，请以仓库当前文件为准。

## 页面归档说明

本仓库中的 HTML 页面均为独立归档，每个页面可单独打开使用。页面内容可能涉及不同主题，建议通过根目录的索引页面浏览全部可用页面。

- 页面文件命名通常反映其内容或用途。
- 页面之间一般不互相依赖（除非有明确引用）。
- 部分页面可能包含外部资源（如 CDN 加载的库），需要网络连接才能完整显示。

## 维护说明

本仓库不定期更新，主要用于个人归档或兴趣项目。维护内容包括：

- 新增页面归档
- 修复页面显示问题
- 更新资源引用或样式

不承诺频繁更新或长期维护。如需使用或引用仓库中的页面内容，请自行评估其适用性。

## 使用方式

1. 克隆或下载本仓库到本地
2. 在浏览器中打开任意 `.html` 文件
3. 或通过 GitHub Pages 等静态托管服务在线浏览（如已配置）
